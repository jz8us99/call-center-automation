# 系统用例文档

## 用户角色
1. **普通用户** - 使用AI智能客服系统的企业主/运营者
2. **系统管理员** - 拥有完全访问权限的系统管理员
3. **员工** - 可被安排预约的员工
4. **客户** - 与AI智能客服交互进行预约/咨询的终端用户
